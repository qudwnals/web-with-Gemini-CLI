## 1. 역할 (Role)
당신은 숙련된 데이터 사이언티스트이자 Python 개발자입니다. 제공된 스타벅스 매장 정보(JSON 포맷)를 바탕으로 데이터 전처리, 탐색적 데이터 분석(EDA), 그리고 시각화를 수행하는 Python 코드를 작성해 주세요.

## 2. 데이터 설명 (Data Description)
데이터는 스타벅스 매장의 상세 정보를 담은 JSON 리스트(`list`)입니다.
주요 컬럼과 데이터 형태는 다음과 같습니다.
- **매장명 (`s_name`)**: 지점 이름
- **위치 정보 (`lat`, `lot`)**: 위도와 경도 (현재 문자열 `String` 타입이므로 변환 필요)
- **주소 (`sido_name`, `gugun_name`, `addr`)**: 시/도, 구/군, 전체 주소
- **오픈일 (`open_dt`)**: `YYYYMMDD` 형식의 문자열
- **테마 정보 (`theme_state`)**: 매장의 특성 태그가 `@` 문자로 연결된 문자열 (예: `T05@T08@P80`)

## 3. 분석 목표 및 단계별 요청 사항

### Step 1: 데이터 전처리 (Preprocessing)
1. JSON 데이터를 **Pandas DataFrame**으로 변환하세요.
2. 데이터 타입을 알맞게 변환하세요:
   - `lat`, `lot` 컬럼을 **실수형(float)**으로 변환하세요.
   - `open_dt` 컬럼을 **날짜형(datetime)**으로 변환하세요.
3. 분석에 불필요한 컬럼은 제외하고, 주요 컬럼 위주로 DataFrame을 재구성하세요.

### Step 2: 탐색적 데이터 분석 (EDA)
1. **지역별 매장 분포**: `sido_name`(시도)별 매장 개수를 집계하여 내림차순으로 정렬하세요.
2. **신규 매장 현황**: 가장 최근에 오픈한 매장 5곳의 이름, 주소, 오픈일을 출력하세요.
3. **테마 분석**:
   - `theme_state` 컬럼을 `@` 기준으로 분리(split)하여 개별 태그 리스트로 만드세요.
   - 전체 매장에서 가장 많이 등장하는 **상위 5개 테마 코드**가 무엇인지 카운트하세요.

### Step 3: 데이터 시각화 (Visualization)
*모든 그래프에는 한글 폰트 깨짐 방지 코드를 포함해야 합니다.*

1. **지역별 분포 시각화**: `sido_name`별 매장 수를 **막대 그래프(Bar Chart)**로 그리세요. (Matplotlib 또는 Seaborn 사용)
2. **지도 시각화**:
   - `folium` 라이브러리를 사용하세요.
   - 지도의 중심은 전체 매장 좌표의 평균값으로 설정하세요.
   - 각 매장의 좌표(`lat`, `lot`)에 마커를 찍고, 마커 클릭 시 매장명(`s_name`)과 주소(`addr`)가 팝업되도록 하세요.
